## Web Scanners Plugin 

Web Scanners plugin will allow to to connect open-source and commercial vulnerability scanners into ArcherySec. Currently we supports below web scanners plugin.  

- [x] [OWASP ZAP](#owasp-zap)
- [x] [Burp Pro](#burp-pro)
- [x] [Arachni](#arachni)


### OWASP ZAP
---------------

### Enable and Get OWASP ZAP scanner API endpoint and Key

<br>

1) Go to preferences in OWASP ZAP proxy tool

2) Open API and clieck on Enabled

3) Copy API Key 
<p align="center">
<img src="../../../assets/images/plugin/zap/owasp-zap-api.png" width="600" height="790" >
</p>


4) Check your IP and port running OWASP ZAP 

<p align="center">
<img src="../../../assets/images/plugin/zap/owasp-zap-api-2.png" width="600" height="790" >
</p>

5) Open ArcherySec Settings 

<p align="center">
<img src="../../../assets/images/plugin/zap/archerysec-1.png" width="600" height="790" >
</p>

6) Edit ZAP Setting and provide all required information 


<p align="center">
<img src="../../../assets/images/plugin/zap/archerysec-2.png" width="600" height="790" >
</p>

7) Test your connection by clicking on Test Connection

<p align="center">
<img src="../../../assets/images/plugin/zap/archerysec-3.png" width="600" height="790" >
</p>


### Burp Pro
---------------

1) Go to Brup Pro User Options > Misc

<p align="center">
<img src="../../../assets/images/plugin/zap/burp.png" width="600" height="790" >
</p>

2) Create new REST API key by clicking on New 

3) Copy key to clipboard

4) Check your burp pro IP and port Proxy > Options 

5) Open ArcherySec settings page

<p align="center">
<img src="../../../assets/images/plugin/zap/burp-1.png" width="600" height="790" >
</p>

6) Edit Burp Settings

7) Provide all required information 

<p align="center">
<img src="../../../assets/images/plugin/zap/burp-2.png" width="600" height="790" >
</p>

8) Save and click on Test Connection to verify connectivity

<p align="center">
<img src="../../../assets/images/plugin/zap/burp-3.png" width="600" height="790" >
</p>
